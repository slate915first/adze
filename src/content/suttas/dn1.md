# DN 1 — Brahmajāla Sutta
*The All-Embracing Net of Views*

62 wrong views the Buddha names and refutes — the complete map of the metaphysical positions a practitioner might fall into. Substantial; advanced; only attempt when ready.

## When to return to this sutta
- [[subcategories/wrong_views|Wrong views to guard]] — The views that trap. The ways the dhamma itself can be grasped wrongly — like the snake seized by the tail. Read these before they catch you.
