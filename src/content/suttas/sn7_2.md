# SN 7.2 — Akkosaka Sutta
*The Abusive Brahmin*

When abused, the Buddha asks the brahmin: "If you offer a gift and it is not received, to whom does it belong?" The same with abuse — if not received, it returns to the giver.

## When to return to this sutta
- [[subcategories/anger|Anger & provocation]] — When someone provokes you. When anger arises in yourself. The suttas on meeting hostility without becoming hostile.
